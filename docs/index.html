Graph: object

An ApiNATOMY model consists of an extended graph with nested subgraphs, called also groups, which depicture the structural composition of human body organs and/or subsystems, with related measurements and annotations. The graph nodes are arranged by the force-directed graph layout that respects constrains on relative positioning of the model elements, specified in the input data by the modeller.

nodes: []

Graph nodes connect separate links into semantically meaningful groups that reflect the organizational composition of body parts and/or flow in physiological processes.

string
Node
links: []

Graph links schematically represent structural elements of the human physiology relevant to the explanation of a certain physiological process: blood vessels, neural connections, pulmonary vessels, etc.

string
Link
lyphs: []

This field lists all lyph definitions in the model. A Lyph provides a schematic explanation of an entity structure, i.e., tissue layers that consistite a certain organ. The lyph is depicted as a flat shape reflecting its topology (tube, bag or a cyst) and positioned along a conveying link, its axis. By rotating the lyph around its axis, we can create a tube (open, semi-closed or closed on both side) that structurally resembles a meaningful physiological element, i.e., a blood vessel or a neuron connection. Each lyph can be composed of other lyphs as its layers or inner elements, and elements that are positioned on its surface can be specified with the help of the lyph's border concept.

Lyph

type: string , x ∈ { link (default) , dashed , semicircle , path , container , force , invisible }

Defines the appearance and behavior of the link in the model graph: link is a straight solid line, dashed is a straight dashed line, semicrcle defines a solid line that looks like half of a circle, path is a solid path with control points defined by the edge bundling algorithm, container is a link with a visual object not affected by the graph forces, force is a link without a visual object used to attract or repel nodes, invisible link is a link with a hidden visual object affected by graph forces that helps to position inner lyph content.
